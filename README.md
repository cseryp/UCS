
# UCS
UCAS course sync tool  
国科大课件资料同步工具

# Requirement

python 2.7+

# Quick Start

- download  
```shell
git clone https://github.com/tianxianbaobao/UCS.git
```
- config  
在config.ini文件中，依次填入SEP系统的用户名，密码，和课件同步位置,如:

		[USER]
		usrname=tianxianbaobao
		passwd=123abc
		savedir=Courses

- run  
```python
python syncc.py
```
